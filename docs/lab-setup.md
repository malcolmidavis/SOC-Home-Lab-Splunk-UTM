# Lab Setup Guide

## Project Goal

The objective of this project was to build a Security Operations Center (SOC) home lab capable of collecting and analyzing system activity using Splunk Enterprise.

---

## Hardware

- Apple MacBook Air (M1)
- 8 GB RAM
- macOS

---

## Software

- UTM
- Ubuntu Server
- Windows 11 ARM
- Splunk Enterprise
- SSH
- macOS Terminal

---

## Environment Configuration

### Ubuntu Server

- Installed Ubuntu Server in UTM.
- Configured networking for remote access.
- Enabled SSH for administration.
- Verified connectivity from the host machine.

### Splunk Enterprise

- Installed Splunk Enterprise on Ubuntu Server.
- Started the Splunk service.
- Accessed the Splunk web interface.
- Completed the initial configuration.
- Verified successful login.

### Windows 11 ARM

- Installed Windows 11 ARM in UTM.
- Used as a test endpoint for future log collection and monitoring.

---

## Result

The completed environment provides a functional virtual SOC lab for practicing SIEM deployment, Linux administration, log analysis, dashboards, alerts, and cybersecurity documentation.
