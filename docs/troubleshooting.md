# Troubleshooting

## Challenges Encountered

### Resource Limitations

The lab was built on an Apple MacBook Air (M1) with 8 GB of RAM. Running multiple virtual machines alongside Splunk Enterprise required careful resource management.

---

### Virtual Machine Performance

Performance varied depending on the number of running virtual machines. Ubuntu Server remained the primary Splunk host throughout the project.

---

### SSH Administration

SSH was used to remotely manage the Ubuntu Server from macOS, providing a more efficient workflow than interacting directly with the VM console.

---

### Windows 11 ARM

Windows 11 ARM was successfully deployed for testing. Additional endpoint monitoring and log forwarding remain areas for future expansion.

---
